## Overview

The CSA Student tier is designed for practitioners who are new to ServiceNow — whether entering the platform for the first time through an IRM lens, or studying toward their Certified System Administrator (CSA) exam as the first step before CIS-RC. It provides the foundational artifacts and structured learning path to move from "new to ServiceNow" to "confident with IRM basics" in a deliberate sequence.

## Key concepts

* **Who it's for** — Anyone beginning their ServiceNow IRM journey: new GRC analysts joining a team that runs ServiceNow, consultants in their first year on the platform, career changers from risk/audit backgrounds entering the ServiceNow ecosystem, and university or bootcamp students pursuing a certification path.
* **Learning path alignment** — The tier's content is sequenced to parallel the CSA exam preparation path and the IRM Foundations module of this handbook. You are not expected to have prior ServiceNow experience to get full value from this tier.
* **PDI-based learning** — This tier assumes you are working in a ServiceNow Personal Developer Instance (PDI), available free from developer.servicenow.com. Every exercise, update set, and template in this tier is designed for a PDI environment.

## What's Included

* **Foundations update set** — The `irm-foundations-v1` update set that installs the baseline IRM configuration demonstrated in the handbook's opening chapters. This is your first hands-on import exercise.
* **CSA certification prep bundle** — Flashcard deck (PDF and Anki format), a 50-question CSA practice exam aligned to the current exam blueprint, and a topic-by-topic study checklist with resource links. Covers all CSA domains: platform navigation, data model, user admin, Business Rules, update sets, and reporting.
* **Basic templates** — Risk Register Starter (XLSX), IRM RACI Matrix (XLSX), and Information Security Policy Template (DOCX) from the template library.
* **Editable architecture diagrams** — The IRM Core Data Model and Three Lines of Defense diagrams in draw.io format, so you can annotate and adapt them as you learn the model.
* **Community Q&A access** — Access to the member community Q&A space to ask questions and follow discussions from more experienced practitioners.

## Recommended Learning Sequence

1. Read the IRM Foundations chapters of this handbook (free, no tier required).
2. Set up a PDI at developer.servicenow.com.
3. Import the `irm-foundations-v1` update set following the import checklist.
4. Work through the IRM Core Data Model diagram — trace a risk from Entity to Control Objective to Indicator.
5. Begin CSA exam prep: complete Now Learning's CSA learning path, then use the practice exam and flashcard deck from this tier to fill gaps.
6. Schedule and pass the CSA exam.
7. Upgrade to Architect Pro to continue into CIS-RC preparation and the full update set library.

{% hint style="info" %}
The CSA exam is a prerequisite for CIS-RC. Complete it before beginning the CIS-RC study plan available in the Architect Pro tier. The platform fundamentals tested on the CSA directly underpin the implementation skills tested on CIS-RC.
{% endhint %}

## Key takeaways

* CSA Student is the right tier if you have less than one year of active ServiceNow experience.
* The CSA exam is the gateway to all ServiceNow implementation specialist certifications — complete it first.
* PDI-based hands-on practice, not passive reading, is the fastest path to platform confidence.
* The community Q&A is a genuine resource — experienced practitioners answer questions, and the discussion archive is searchable.
* When you have passed CSA and are ready to implement IRM for an organization, upgrade to Architect Pro.
